# Environment Information

The project was developed and tested using **Java 17**, **Maven**, **Docker**, and **Testcontainers**.
The CI pipeline is implemented using **GitHub Actions** and performs:
- Unit tests
- Integration tests
- UI tests (Swing with Xvfb)
- Code coverage analysis using **JaCoCo**
- Coverage reporting via **Coveralls**
- Static code analysis and quality gate enforcement via **SonarCloud**

The project follows **strict Test-Driven Development (TDD)** practices and ensures:
- High test coverage
- Zero critical issues
- Zero vulnerabilities
- Passed quality gate on SonarCloud
